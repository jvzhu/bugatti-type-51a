## 🇺🇸 English Documentation

### Overview

**Bugatti Type 51A: Automotive Design Archive** is a comprehensive, interactive documentation platform celebrating one of automotive history's most iconic vehicles. This project showcases the legendary Bugatti Type 51A through design evolution, technical specifications, racing history, and historical context.

### 🎯 Features

- **🎨 Interactive Design Timeline** — Visual evolution from 1920s to 1960s
- **🔧 Technical Specifications Database** — Searchable, sortable technical data
- **🖼️ Photo Gallery** — High-resolution imagery with lightbox viewer
- **🏁 Racing History** — Grand Prix victories and driver profiles
- **🌙 Dark Mode Support** — Elegant theme switching
- **🌍 Multilingual Interface** — English, Chinese, Japanese, French, German, Spanish, Italian
- **📱 Responsive Design** — Mobile-first, works on all devices
- **♿ Accessibility** — WCAG AA compliant
- **⚡ Performance Optimized** — Fast loading and smooth interactions

### 🛠️ Tech Stack

```
Frontend:    React 18 with Hooks
Styling:     CSS3 (modular, variables-based)
Data:        JSON (specifications, racing history, timeline)
Deployment:  GitHub Pages
Build:       Create React App / Vite
```

### 📦 Installation

```bash
# Clone the repository
git clone https://github.com/jvzhu/bugatti-type-51a.git
cd bugatti-type-51a

# Install dependencies
npm install

# Start development server
npm start

# Build for production
npm run build
```

## 🇨🇳 中文简体文档

### 项目概述

**布加迪Type 51A：汽车设计档案** 是一个全面的交互式文档平台，致力于庆祝汽车历史上最具标志性的车型之一。本项目通过设计演进、技术规格、赛车历史和历史背景展示传奇的布加迪Type 51A。

- **🎨 交互式设计时间线** — 从1920年代到1960年代的视觉演进

- **🌙 深色模式支持** — 优雅的主题切换
- **🌍 多语言界面** — 英文、中文、日文、法文、德文、西班牙文、意大利文
- **📱 响应式设计** — 移动优先，兼容所有设备
- **♿ 无障碍访问** — WCAG AA 合规
